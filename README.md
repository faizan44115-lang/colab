# Customer Segmentation and Recommendation System

## Project Overview
This project performs customer segmentation using RFM Analysis and K-Means Clustering.

It also recommends products using Item-Based Collaborative Filtering and Cosine Similarity.

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- RFM Analysis
- K-Means Clustering
- Customer Segmentation
- Product Recommendation System
- Streamlit Deployment

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Matplotlib
- Seaborn

pip install -r requirements.txt
